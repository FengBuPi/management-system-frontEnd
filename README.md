# 个人财务管理系统-前后端分离版本(React+Redux+tailwind+ Nestjs+Swagger+Mysql)

## Description

**技术栈：** React+tsx+Redux+tailwind+ Nestjs+Swagger+typeOrm-Mysql

## 安装依赖

```bash
$ pnpm install
```

## 运行程序

```bash
$ pnpm run dev
```
